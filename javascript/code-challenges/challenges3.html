301 Code Challenges: Collection Three

Challenge #1:

Write a function called addTwo that takes in an array and adds two to every value using a for loop. Place the new value in a new array. Return the new array.

Challenge #2:

Write a function named typeNum that, given an array as input, uses filter to return an array containing only the numbers. For example, typeNum([1, 'bob' ,3]) returns [1,3].

Challenge #3:

Write a function named containsAnd that, given an array of strings as input, uses filter to return an array containing only strings that contain 'and' within the string. For example, containsAnd(['panda', 'ran', 'and']) returns ['panda', 'and'].

Challenge #4:

Write a function named oddValues that, given an array of integers as input, uses filter to return an array containing only the odd integers. For example, oddValues([1,2,3]) returns [1,3].

Challenge #5:

Write a function named notInFirstArray that, given two arrays as input, uses filter to return an array of all the elements in the second array that are not included in the first array. For example, notInFirstArray([1,2,3], [1,2,3,4]) returns [4].

For example, getBaseStatGreaterThan(snorlaxData.stats, 50) will return an array containing the 'special-defense' and 'special-attack' objects.

For example, getStatName(snorlaxData.stats, 50) will return ['special-defense', 'special-attack'].

Stretch Goal (8): Not attempted yet.

Write a function named getCharactersWithoutChildren that, given the array of characters, below, uses filter to return an array of all characters without children.

Stretch Goal (9): Not attempted yet.

For example: evenOddNumericValues(['Gregor', 2, 4, 1]) returns ['even', 'even', 'odd'].
